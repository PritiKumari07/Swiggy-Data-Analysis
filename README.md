# Swiggy-Data-Analysis-Dashboard

This project presents an interactive data analysis dashboard built using food delivery data inspired by Swiggy. The goal of this project is to analyze sales performance, customer ratings, order trends, and geographic distribution of orders to derive useful business insights.
The dashboard helps understand how different food categories, cities, and time periods impact sales and customer engagement.

## Project Overview

This data analysis project focuses on:

* Understanding sales performance across different states and cities.
* Analyzing customer ratings and order patterns.
* Identifying high-performing food categories.
* Tracking monthly, weekly, and daily sales trends.
* Discovering top restaurants and locations contributing to revenue.
* The dashboard provides a clear visual representation of the dataset to support data-driven decision making.

## Key Insights from the Dashboard

* Total sales recorded: ₹53.01M
* Average customer rating: 4.34
* Average order value: ₹268.51
* Total orders: 197K+
* Veg orders contribute around 63% of sales
* Bengaluru is the top-performing city in sales

## Tools & Technologies Used

* Microsoft Excel (Dashboard Creation)
* Data Cleaning & Transformation
* Data Visualization
* Pivot Tables
* Data Analysis

## Project Outcome

This project demonstrates how raw data can be transformed into meaningful insights using data analysis techniques and dashboard visualization. It also reflects practical skills in business analysis, identifying trends, and presenting data in a clear and impactful way.
